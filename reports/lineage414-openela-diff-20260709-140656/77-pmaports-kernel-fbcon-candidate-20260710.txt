pmaports kernel fbcon candidate - 2026-07-10
================================================

Reason
------

The screen can show DRM userspace text, but the validated `215005` kernel does
not have the kernel-console pieces needed for real boot text:

```text
# CONFIG_VT is not set
# CONFIG_DRM_FBDEV_EMULATION is not set
# CONFIG_FB_SIMPLE is not set
```

The local pmaports `linux-oneplus-hotdog-lineage414` package already carries
the intended console config:

```text
CONFIG_VT=y
CONFIG_VT_CONSOLE=y
CONFIG_DRM_FBDEV_EMULATION=y
CONFIG_FB_SIMPLE=y
CONFIG_FRAMEBUFFER_CONSOLE=y
```
