  eth* interfaces no longer count as USB success.

Why:

  The known-good pmOS NCM path uses usb0 with carrier=1 and operstate=up. A
  generic eth* interface could be unrelated to the USB gadget path on another
  boot environment and could incorrectly stop the rescue reboot. This candidate
  keeps the useful screen diagnostics while reducing that false-positive risk.
