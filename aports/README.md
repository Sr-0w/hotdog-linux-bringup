# Local aport snapshots

This directory contains small, curated pmaports package snapshots that are
needed to reproduce the current hotdog bring-up state without copying the full
`src/postmarketos/pmaports-sm8150` checkout.

The canonical working tree during local development is still:

```text
src/postmarketos/pmaports-sm8150
```

Snapshot contents are updated only when a local aport becomes important enough
for resume-on-another-machine work.
